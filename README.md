# week

週単位で思考を整理し、振り返りと次週の目標設定をサポートするiOSアプリです。日記が続かない人でも、自分のペースで記録を積み重ねて自己成長を実感できます。

## アプリ概要
<img width="749" height="553" alt="image" src="https://github.com/user-attachments/assets/86f16f62-0671-4c02-80a7-a7c84bc3287f" />

- 日々の思考を気軽に書き留め、週末に振り返りと次週の目標設定を行う。
- 記録はローカルに保存され、プライバシーを守りながら自己理解を深める。
- 穏やかなトーンのUIとシンプルな操作で継続しやすさを重視。

> 流れる日々を記録する / 少しでも前に進む / 自分と比べる

## App Store向けメッセージ

### こんな方におすすめ
- 日記が続かない方
- 他人と比べて疲れてしまう方
- 思考を整理したい方
- 目標達成のモチベーションを上げたい方

### Weekの特徴
1. **柔軟な記録**  
   毎日書く必要はなく、自分のペースでメモを残せます。思いついたときに素早く記録できるシンプルな操作性。
2. **週間振り返り**  
   日曜日に一週間を振り返りながら、来週の目標や重点を設定できます。
3. **アーカイブ機能**  
   過去の記録を週単位で見返し、他人ではなく「過去の自分」と比較して成長を確認。
4. **ダークモード対応**  
   システム設定に連動した夜間モードで、目に優しく快適に利用できます。
5. **データの安全性**  
   記録はすべてローカル保存。ネットワークに依存せずプライバシーを保護します。
6. **直感的なUI**  
   穏やかな色合いと小さなアニメーションで、リラックスしながら使える体験を提供します。

## ターゲットユーザー

**コアユーザー像**
- 20〜30代の学生・社会人
- 忙しい日々を送りながらも自己成長への関心が高い
- スマートフォンアプリの利用に慣れ、テクノロジーへの抵抗がない

**課題とモチベーション**
- 日記や振り返りを継続できず、時間があっという間に過ぎてしまう感覚がある
- 日々の生活に流されず、自分の変化を実感したい
- 短時間で完結するコンテンツに流れてしまい、自己投資の時間を確保しづらい

**期待する価値**
- 思考のパターンを把握し、生活の質を高めたい
- 小さな目標設定と達成を繰り返して自己効力感を育てたい
- 週末の振り返り習慣づくりを通じて、自分に合ったペースを見つけたい

## 使用シーン

- 週末に一週間を振り返り、次週の目標やテーマを整理するとき
- 日中のスキマ時間に思いついたアイデアや感情をメモするとき
- 過去の週を振り返り、成長や変化を確認するとき

## デザイン哲学

- **構造化**: 思考を箇条書きで整理しやすくする。
- **柔軟性**: ユーザーが自由な粒度で記録し、振り返り方法を選べる。
- **有機的**: 穏やかなアニメーションで生活のリズムに寄り添う。

## アーキテクチャと技術要件

- **アプリ構成**: SwiftUIによるiOSクライアント
- **データ保存**: Core Dataを用いたローカルデータベース
- **同期/共有**: 現状はローカルのみで完結
- **バージョン管理**: GitHub
- **対応プラットフォーム**: iOS

## データモデル

**思考カード (ThoughtCard)**
- `id`: UUID
- `content`: String
- `date`: Date
- `items`: [String]

**週の記録 (WeeklyRecord)**
- `id`: UUID
- `startDate`: Date
- `endDate`: Date
- `thoughts`: [ThoughtCard]
- `reflection`: String
- `goal`: String

### Core Dataエンティティ
- `ThoughtCardEntity`: ThoughtCardを永続化
- `WeeklyRecordEntity`: 週次の記録と目標を永続化
