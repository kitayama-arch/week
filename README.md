- **weekアプリ要件定義書**
    
    ### appstore
    【こんな方におすすめ】
    - 日記が続かない方
    - 他人と比べて疲れてしまう方
    - 思考を整理したい方
    - 目標達成のモチベーションを上げたい方

    【Weekの特徴】
    1. 柔軟な記録: 
    毎日記録する必要はなく、自分のペースで思考を記録できます。もちろん一日に何個もメモすることも可能です。
    思いついたときにすぐに記録できる、シンプルな操作性を重視しています。

    2. 週間振り返り: 
    日曜日になると、今週の振り返りができます。今週の思考を振り返りながら、来週の目標や振り返りを設定できます。

    3. アーカイブ機能: 
    過去の記録を週単位で閲覧し、自身の成長を可視化します。
    他人と比べるのではなく、自分がどう成長できているかを大切にするお手伝いをします。

    4. ダークモード対応: 
    目に優しい夜間モードで、快適に使用できます。
    システム設定に連動して自動的に切り替わります。

    5. データの安全性: 
    すべてのデータはローカルに保存され、プライバシーが保護されます。

    6. 直感的なUI: 
    シンプルで使いやすいインターフェースにより、複雑な操作は不要です。
    穏やかで落ち着いた色合いのデザインで、リラックスして使用できます。  
    
    ### 概要
    
    weekは、ユーザーがぼーっと思考を書き連ね、週単位で振り返りと目標設定を行うアプリです。記録をアーカイブし、次週の目標を設定することで、日記が続かない人でも自己成長を促進します。
    
    流れる日々を記録する
    
    少しでも前に進む
    
    自分と比べる
    
    ### ターゲットユーザー
    
    - **日記が続かない人**: 週単位で思考を整理するためのツールとして利用。
    - **自己成長を目指す人**: 自分の中で幸せを作りたい人が、次週の目標を設定し、それを達成するためのモチベーションツールとして利用。
    
    ### ターゲットユーザーの詳細プロフィール
    
    - 年齢層：20〜30代
    - 職業・生活スタイル：
      - 学生や社会人
      - 日常的に決まったルーティンを繰り返している
      - 忙しい日々を送っている
    - テクノロジーへの親和性：
      - テクノロジーに興味があり、好んで使用する
      - スマートフォンやアプリの利用に慣れている
    - 抱えている問題・課題：
      - 時間の経過が早く感じられ、気づくと一週間が過ぎている
      - 日記を続けることが難しい
      - 日々の生活に対する意識が薄れがち
    - アプリ使用の動機：
      - 日々の思考や出来事を簡単に記録したい
      - 週単位で自分の生活を振り返りたい
      - 自己成長や改善のきっかけを得たい
    - アプリの使用シーン：
      - 思いついたアイデアや考えをすぐに記録する
      - 週末に一週間の振り返りを行う
      - 空き時間にアプリを開いて過去の記録を見返す
    - 価値観・興味関心：
      - 自己成長に興味がある
      - 効率的な時間の使い方を模索している
      - 生活の質を向上させたいと考えている
    - 現在の行動パターン：
      - SNSでリールなどを見て、無為に時間を過ごすことがある
      - 短い時間で完結するコンテンツを好む傾向がある
    - アプリの特徴に対する反応：
      - 週単位の記録と振り返りという新しいアプローチに興味を持つ
      - 日記よりも気軽に続けられそうだと感じる
    - 期待する成果：
      - 自分の生活や思考のパターンを把握したい
      - 小さな目標設定と達成を繰り返し、自己効力感を高めたい
      - 時間の使い方を意識し、より充実した日々を送りたい
    
    ### 使用シーン
    
    - **週末のリフレクションタイム**: 一週間を振り返り、次週の目標を設定。
    - **日常の合間**: 思考を整理するためのツールとして利用。
    
    ### デザイン哲学
    
    - **構造化**: 思考を箇条書きで構造化。
    - **柔軟性**: ユーザーが自由に思考を記録し、目標を設定。
    - 有機的:小さなアニメーションで生活に則したもの
    
    ### アーキテクチャ
    
    - **クライアントサイド**: SwiftUIを用いたiOSアプリ
    - **データ保存**: Core Dataを利用したローカルデータベース
    - **バージョン管理**: GitHubを利用
    
    
    ### データモデル
    
    - **思考カード (ThoughtCard)**
        - `id`: UUID
        - `content`: String
        - `date`: Date
        - `items`: [String]
    - **週の記録 (WeeklyRecord)**
        - `id`: UUID
        - `startDate`: Date
        - `endDate`: Date
        - `thoughts`: [ThoughtCard]
        - `reflection`: String
        - `goal`: String
    
    ### データ管理
    
    - **Core Data**
        - **エンティティ**
            - `ThoughtCardEntity`
                - `id`: UUID
                - `content`: String
                - `date`: Date
                - `items`: [String]
            - `WeeklyRecordEntity`
                - `id`: UUID
                - `startDate`: Date
                - `endDate`: Date
                - `thoughts`: [ThoughtCardEntity]
                - `reflection`: String
                - `goal`: String
